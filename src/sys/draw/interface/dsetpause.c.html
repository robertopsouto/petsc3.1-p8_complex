  9: /*@
 10:    PetscDrawSetPause - Sets the amount of time that program pauses after 
 11:    a PetscDrawPause() is called. 

 13:    Collective on PetscDraw

 15:    Input Parameters:
 16: +  draw   - the drawing object
 17: -  lpause - number of seconds to pause, -1 implies until user input

 19:    Level: intermediate

 21:    Note:
 22:    By default the pause time is zero unless the -draw_pause option is given 
 23:    during PetscDrawOpenX().

 25:    Concepts: drawing^waiting

 27: .seealso: PetscDrawGetPause(), PetscDrawPause()
 28: @*/
 29: PetscErrorCode  PetscDrawSetPause(PetscDraw draw,PetscReal lpause)
 30: {
 33:   draw->pause = lpause;
 34:   return(0);
 35: }
