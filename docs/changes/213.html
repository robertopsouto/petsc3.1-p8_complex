CHANGES in PETSc 2.1.3

General:

This is primarily a bug-fix release and minor internal updates. Some of the fixes include

  • win32fe updates for better build support on Windows 98 machines, with borland compilers, and with cygwin paths.
  • fixes to SPOOLES interfaces
  • updates to DA  - which delays allocation of internal objects untill they are actually needed.
  • configure fixes for linux
