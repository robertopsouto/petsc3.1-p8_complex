TSGLInitializePackage

 This function initializes everything in the TSGL package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to TSCreate_GL() when using static libraries.

Synopsis

#include "petscts.h"   
PetscErrorCode PETSCTS_DLLEXPORT TSGLInitializePackage(const char path[])

Input Parameter

path - The dynamic library path, or PETSC_NULL

Keywords

TS, TSGL, initialize, package

See Also

PetscInitialize()
