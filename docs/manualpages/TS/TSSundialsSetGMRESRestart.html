TSSundialsSetGMRESRestart

 Sets the dimension of the Krylov space used by GMRES in the linear solver in SUNDIALS. SUNDIALS DOES NOT use restarted GMRES so this is ALSO the maximum number of GMRES steps that will be used.

Synopsis

#include "petscts.h" 
PetscErrorCode PETSCTS_DLLEXPORT TSSundialsSetGMRESRestart(TS ts,int restart)
Collective on TS

Input parameters

ts - the time-step context
restart - number of direction vectors (the restart size).

Keywords

GMRES, restart
