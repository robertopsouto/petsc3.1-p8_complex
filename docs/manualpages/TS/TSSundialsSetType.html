TSSundialsSetType

 Sets the method that Sundials will use for integration.

Synopsis

#include "petscts.h" 
PetscErrorCode PETSCTS_DLLEXPORT TSSundialsSetType(TS ts,TSSundialsLmmType type)
Collective on TS

Input parameters

ts - the time-step context
type - one of SUNDIALS_ADAMS or SUNDIALS_BDF

Keywords

Adams, backward differentiation formula
