PCApplyTranspose

 Applies the transpose of preconditioner to a vector.

Synopsis

#include "petscksp.h" 
PetscErrorCode PETSCKSP_DLLEXPORT PCApplyTranspose(PC pc,Vec x,Vec y)
Collective on PC and Vec

Input Parameters

pc - the preconditioner context
x - input vector

Output Parameter

y -output vector
