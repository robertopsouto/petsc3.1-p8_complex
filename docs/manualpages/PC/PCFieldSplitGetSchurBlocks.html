PCFieldSplitGetSchurBlocks

 Gets the all matrix blocks for the Schur complement

Synopsis

#include "petscpc.h" 
PetscErrorCode PETSCKSP_DLLEXPORT PCFieldSplitGetSchurBlocks(PC pc,Mat *A,Mat *B,Mat *C, Mat *D)
Collective on KSP

Input Parameter

pc -the preconditioner context

Output Parameters

A - the (0,0) block
B - the (0,1) block
C - the (1,0) block
D - the (1,1) block
