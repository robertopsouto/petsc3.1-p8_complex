PCInitializePackage

 This function initializes everything in the PC package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to PCCreate() when using static libraries.

Synopsis

PetscErrorCode PETSCKSP_DLLEXPORT PCInitializePackage(const char path[])

Input Parameter

path - The dynamic library path, or PETSC_NULL

Keywords

PC, initialize, package

See Also
