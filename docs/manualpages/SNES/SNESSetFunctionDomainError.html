SNESSetFunctionDomainError

 tells SNES that the input vector to your FormFunction is not in the functions domain. For example, negative pressure.

Synopsis

#include "petscsnes.h"  
PetscErrorCode PETSCSNES_DLLEXPORT SNESSetFunctionDomainError(SNES snes)
Collective on SNES

Input Parameters

SNES -the SNES context

Keywords

SNES, view
