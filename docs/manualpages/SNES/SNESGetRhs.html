SNESGetRhs

 Gets the vector for solving F(x) = rhs. If rhs is not set it assumes a zero right hand side.

Synopsis

#include "petscsnes.h"  
PetscErrorCode PETSCSNES_DLLEXPORT SNESGetRhs(SNES snes,Vec *rhs)
Collective on SNES

Input Parameter

snes -the SNES context

Output Parameter

rhs -the right hand side vector or PETSC_NULL if the right hand side vector is null

Keywords

SNES, nonlinear, get, function, right hand side

See Also

SNESGetSolution(), SNESGetFunction(), SNESComputeFunction(), SNESSetJacobian(), SNESSetFunction()
