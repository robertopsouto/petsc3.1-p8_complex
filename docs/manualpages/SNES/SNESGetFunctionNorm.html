SNESGetFunctionNorm

 Gets the norm of the current function that was set with SNESSSetFunction().

Synopsis

#include "petscsnes.h"  
PetscErrorCode PETSCSNES_DLLEXPORT SNESGetFunctionNorm(SNES snes,PetscReal *fnorm)
Collective on SNES

Input Parameter

snes -SNES context

Output Parameter

fnorm -2-norm of function

Keywords

SNES, nonlinear, get, function, norm
