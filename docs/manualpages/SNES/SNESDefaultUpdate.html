SNESDefaultUpdate

 The default update function which does nothing.

Synopsis

#include "petscsnes.h"  
PetscErrorCode PETSCSNES_DLLEXPORT SNESDefaultUpdate(SNES snes, PetscInt step)
Not collective

Input Parameters

snes -The nonlinear solver context
