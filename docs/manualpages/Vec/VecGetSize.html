VecGetSize

 Returns the global number of elements of the vector.

Synopsis

#include "petscvec.h" 
PetscErrorCode  VecGetSize(Vec x,PetscInt *size)
Not Collective

Input Parameter

x -the vector

Output Parameters

size -the global length of the vector
