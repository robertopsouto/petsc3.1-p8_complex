VecValid

 Checks whether a vector object is valid.

Synopsis

#include "petscvec.h" 
PetscErrorCode PETSCVEC_DLLEXPORT VecValid(Vec v,PetscTruth *flg)
Not Collective

Input Parameter

v -the object to check

Output Parameter

flg -flag indicating vector status, either PETSC_TRUE if vector is valid, or PETSC_FALSE otherwise.
