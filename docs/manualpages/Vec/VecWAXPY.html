VecWAXPY

 Computes w = alpha x + y.

Synopsis

#include "petscvec.h" 
PetscErrorCode  VecWAXPY(Vec w,PetscScalar alpha,Vec x,Vec y)
Collective on Vec

Input Parameters

alpha - the scalar
x, y - the vectors

Output Parameter

w -the result

Notes: Neither the vector x or y can be the same as vector w

See Also

VecAXPY(), VecAYPX(), VecAXPBY()
