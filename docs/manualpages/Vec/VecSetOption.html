VecSetOption

 Sets an option for controling a vector's behavior.

Synopsis

#include "petscvec.h" 
PetscErrorCode PETSCVEC_DLLEXPORT VecSetOption(Vec x,VecOption op,PetscTruth flag)
Collective on Vec

Input Parameter

x - the vector
op - the option
flag - turn the option on or off
