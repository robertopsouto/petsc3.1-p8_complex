VecMAXPY

 Computes y = y + sum alpha[j] x[j]

Synopsis

#include "petscvec.h" 
PetscErrorCode  VecMAXPY(Vec y,PetscInt nv,const PetscScalar alpha[],Vec x[])
Collective on Vec

Input Parameters

nv - number of scalars and x-vectors
alpha - array of scalars
y - one vector
x - array of vectors

See Also

VecAXPY(), VecWAXPY(), VecAYPX()
