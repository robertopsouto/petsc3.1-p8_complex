ISView

 Displays an index set.

Synopsis

#include "petscis.h" 
PetscErrorCode PETSCVEC_DLLEXPORT ISView(IS is,PetscViewer viewer)
Collective on IS

Input Parameters

is - the index set
viewer - viewer used to display the set, for example PETSC_VIEWER_STDOUT_SELF.
