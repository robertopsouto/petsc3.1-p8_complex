DMMGGetSNES

 Gets the SNES object (nonlinear solver) for the finest level

Synopsis

SNES DMMGGetSNES(DMMG *dmmg)
Not Collective

Input Parameter

dmmg -DMMG solve context

Notes: If this is a linear problem (i.e. DMMGSetKSP() was used) then this returns PETSC_NULL
