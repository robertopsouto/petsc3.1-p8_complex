DMMGSetOptionsPrefix

 Sets the prefix used for the solvers inside a DMMG

Synopsis

#include "petscda.h"   
#include "petscksp.h"  
#include "petscmg.h"   
#include "petscdmmg.h" 
#include "petscpc.h"   
PetscErrorCode PETSCSNES_DLLEXPORT DMMGSetOptionsPrefix(DMMG *dmmg,const char prefix[])
Collective on MPI_Comm

Input Parameters

dmmg - the DMMG object created with DMMGCreate()
prefix - the prefix string
