DAGetISLocalToGlobalMappingBlck

 Accesses the local-to-global mapping in a DA.

Synopsis

#include "petscda.h"   
PetscErrorCode PETSCDM_DLLEXPORT DAGetISLocalToGlobalMappingBlck(DA da,ISLocalToGlobalMapping *map)
Not Collective

Input Parameter

da -the distributed array that provides the mapping

Output Parameter

ltog -the mapping

Notes

This mapping can them be used by VecSetLocalToGlobalMappingBlock() or MatSetLocalToGlobalMappingBlock().

Essentially the same data is returned in the form of an integer array with the routine DAGetGlobalIndices().

Keywords

distributed array, destroy
