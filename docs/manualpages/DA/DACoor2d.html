DACoor2d

 Structure for holding 2d (x and y) coordinates.

Sample Usage

DACoor2d **coors; Vec vcoors; DA cda;

DAGetCoordinates(da,&vcoors); DAGetCoordinateDA(da,&cda); DAVecGetArray(cda,vcoors,&coors); DAGetCorners(cda,&mstart,&nstart,0,&m,&n,0) for (i=mstart; i<mstart+m; i++) { for (j=nstart; j<nstart+n; j++) { x = coors[j][i].x; y = coors[j][i].y; ...... } } DAVecRestoreArray(dac,vcoors,&coors);
