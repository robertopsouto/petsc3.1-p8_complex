DACreateLocalVector

 Creates a Seq PETSc vector that may be used with the DAXXX routines.

Synopsis

#include "petscda.h"   
PetscErrorCode PETSCDM_DLLEXPORT DACreateLocalVector(DA da,Vec* g)
Not Collective

Input Parameter

da -the distributed array

Output Parameter

g -the local vector

Note

The output parameter, g, is a regular PETSc vector that should be destroyed with a call to VecDestroy() when usage is finished.

Keywords

distributed array, create, local, vector
