DMLocalToGlobal

 updates global vectors from local vectors

Synopsis

#include "petscda.h"     
PetscErrorCode PETSCDM_DLLEXPORT DMLocalToGlobal(DM dm,Vec g,InsertMode mode,Vec l)
Collective on DM

Input Parameters

dm - the DM object
g - the global vector
mode - INSERT_VALUES or ADD_VALUES
l - the local vector

See Also
