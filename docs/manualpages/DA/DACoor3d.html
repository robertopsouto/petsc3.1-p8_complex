DACoor3d

 Structure for holding 3d (x, y and z) coordinates.

Sample Usage

DACoor3d ***coors; Vec vcoors; DA cda;

DAGetCoordinates(da,&vcoors); DAGetCoordinateDA(da,&cda); DAVecGetArray(cda,vcoors,&coors); DAGetCorners(cda,&mstart,&nstart,&pstart,&m,&n,&p) for (i=mstart; i<mstart+m; i++) { for (j=nstart; j<nstart+n; j++) { for (k=pstart; k<pstart+p; k++) { x = coors[k][j][i].x; y = coors[k][j][i].y; z = coors[k][j][i].z; ...... } } DAVecRestoreArray(dac,vcoors,&coors);
