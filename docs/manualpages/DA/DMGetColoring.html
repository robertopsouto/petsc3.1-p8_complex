DMGetColoring

 Gets coloring for a DA or DMComposite

Synopsis

#include "petscda.h"     
PetscErrorCode PETSCDM_DLLEXPORT DMGetColoring(DM dm,ISColoringType ctype,const MatType mtype,ISColoring *coloring)
Collective on DM

Input Parameter

dm - the DM object
ctype - IS_COLORING_GHOSTED or IS_COLORING_GLOBAL
matype - either MATAIJ or MATBAIJ

Output Parameter

coloring -the coloring

See Also

DMDestroy(), DMView(), DMCreateGlobalVector(), DMGetInterpolation(), DMGetMatrix()
