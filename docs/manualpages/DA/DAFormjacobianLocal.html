DAFormjacobianLocal

 This is a universal Jacobian evaluation routine for a local DA function.

Synopsis

#include "petscda.h"   
PetscErrorCode PETSCDM_DLLEXPORT DAFormJacobianLocal(DA da, DALocalFunction1 func, Vec X, Mat J, void *ctx)
Collective on DA

Input Parameters

da - the DA context
func - The local function
X - input vector
J - Jacobian matrix
ctx - A user context
