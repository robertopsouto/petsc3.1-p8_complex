DAFormFunctionLocalGhost

 This is a universal function evaluation routine for a local DA function, but the ghost values of the output are communicated and added.

Synopsis

#include "petscda.h"   
PetscErrorCode PETSCDM_DLLEXPORT DAFormFunctionLocalGhost(DA da, DALocalFunction1 func, Vec X, Vec F, void *ctx)
Collective on DA

Input Parameters

da - the DA context
func - The local function
X - input vector
F - function vector
ctx - A user context
