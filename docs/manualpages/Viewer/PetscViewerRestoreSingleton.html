PetscViewerRestoreSingleton

 Restores a new PetscViewer obtained with PetscViewerGetSingleton().

Synopsis

#include "petscviewer.h" 
PetscErrorCode PETSC_DLLEXPORT PetscViewerRestoreSingleton(PetscViewer viewer,PetscViewer *outviewer)
Collective on PetscViewer

Input Parameters

viewer - the PetscViewer to be duplicated
outviewer - new PetscViewer

Notes: Call PetscViewerGetSingleton() to get this PetscViewer, NOT PetscViewerCreate()

See Also

PetscViewerSocketOpen(), PetscViewerASCIIOpen(), PetscViewerDrawOpen(), PetscViewerGetSingleton()
