PetscDrawHGGetDraw

Synopsis

#include "petscsys.h" 
PetscErrorCode PETSC_DLLEXPORT PetscDrawHGGetDraw(PetscDrawHG hist, PetscDraw *win)
Not Collective, PetscDraw is parallel if PetscDrawHG is parallel
