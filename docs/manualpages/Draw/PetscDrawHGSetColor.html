PetscDrawHGSetColor

 Sets the color the bars will be drawn with.

Synopsis

#include "petscsys.h" 
PetscErrorCode PETSC_DLLEXPORT PetscDrawHGSetColor(PetscDrawHG hist, int color)
Not Collective (ignored except on processor 0 of PetscDrawHG)

Input Parameters

hist - The histogram context
color - one of the colors defined in petscdraw.h or PETSC_DRAW_ROTATE to make each bar a different color
