PetscDrawLGSPDraw

Synopsis

#include "petscsys.h" 
PetscErrorCode PETSC_DLLEXPORT PetscDrawLGSPDraw(PetscDrawLG lg,PetscDrawSP spin)
Not Collective,but ignored by all processors except processor 0 in PetscDrawLG

Input Parameter

lg -the line graph context
