PetscGlobalMin

 Computes the minimum value over several processors

Synopsis

#include "petscsys.h"   
PetscErrorCode PETSC_DLLEXPORT PetscGlobalMin(PetscReal* local,PetscReal* result,MPI_Comm comm)
Collective on MPI_Comm

Input Parameters

local - the local value
comm - the processors that find the minimum

Output Parameter

result -the minimum value

Notes

These functions are to be used inside user functions that are to be processed with ADIC. PETSc will automatically provide differentiated versions of these functions

See Also

PetscGlobalMax(), PetscGlobalSum()
