PetscGetVersion

 Gets the PETSc version information in a string.

Input Parameter

len -length of the string

Output Parameter

version -version string

Usage

char version[256]; ierr = PetscGetVersion(version,256);CHKERRQ(ierr)

Fortran Note

This routine is not supported in Fortran.
