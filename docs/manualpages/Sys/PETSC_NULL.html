PETSC_NULL

 standard way of passing in a null or array or pointer

Notes: accepted by many PETSc functions to not set a parameter and instead use some default

This macro does not exist in Fortran; you must use PETSC_NULL_INTEGER, PETSC_NULL_DOUBLE_PRECISION etc

See Also

PETSC_DECIDE, PETSC_DEFAULT, PETSC_IGNORE, PETSC_DETERMINE
