PetscPolymorphicSubroutine

 allows defining a C++ polymorphic version of a PETSc function that remove certain optional arguments for a simplier user interface

Synopsis

PetscPolymorphicSubroutine(Functionname,(arguments of C++ function),(arguments of C function))
Not collective

Example

PetscPolymorphicSubroutine(VecNorm,(Vec x,PetscReal *r),(x,NORM_2,r)) generates the new routine PetscErrorCode VecNorm(Vec x,PetscReal *r) = VecNorm(x,NORM_2,r)
