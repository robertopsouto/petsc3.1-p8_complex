PetscStrcat

 Concatenates a string onto a given string

Synopsis

#include "petscsys.h"   
PetscErrorCode  PetscStrcat(char s[],const char t[])
Not Collective

Input Parameters

s - string to be added to
t - pointer to string to be added to end
