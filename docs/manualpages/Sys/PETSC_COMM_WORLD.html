PETSC_COMM_WORLD

 the equivalent of the MPI_COMM_WORLD communicator which represents all the processs that PETSc knows about.

Notes: By default PETSC_COMM_WORLD and MPI_COMM_WORLD are identical unless you wish to run PETSc on ONLY a subset of MPI_COMM_WORLD. In that case create your new (smaller) communicator, call it, say comm, and set PETSC_COMM_WORLD = comm BEFORE calling PetscInitialize()
