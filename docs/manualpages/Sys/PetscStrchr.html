PetscStrchr

 Locates first occurance of a character in a string

Synopsis

#include "petscsys.h"   
PetscErrorCode PETSC_DLLEXPORT PetscStrchr(const char a[],char b,char *c[])
Not Collective

Input Parameters

a - pointer to string
b - character

Output Parameter

c -location of occurance, PETSC_NULL if not found
