PetscObjectPublish

 Publish an object

Synopsis

#include "petscsys.h"   
PetscErrorCode  PetscObjectPublish(PetscObject obj)
Collective on PetscObject

Input Parameters

obj -the Petsc variable Thus must be cast with a (PetscObject), for example, PetscObjectSetName((PetscObject)mat,name);

Notes: Not currently used
