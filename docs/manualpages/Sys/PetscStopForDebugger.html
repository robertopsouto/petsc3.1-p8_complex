PetscStopForDebugger

 Prints a message to the screen indicating how to attach to the process with the debugger and then waits for the debugger to attach.

Synopsis

#include "petscsys.h"   
PetscErrorCode  PetscStopForDebugger(void)
Not Collective

See Also
