PetscOptionsAllUsed

 Returns a count of the number of options in the database that have never been selected.

Synopsis

#include "petscsys.h"   
PetscErrorCode PETSC_DLLEXPORT PetscOptionsAllUsed(int *N)
Not Collective

Output Parameter

N -count of options not used
