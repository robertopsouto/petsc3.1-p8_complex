PetscBagSetName

 Sets the name of a bag of values

Synopsis

#include "petscbag.h"   
PetscErrorCode PetscBagSetName(PetscBag bag, const char *name, const char *help)
Not Collective

Input Parameters

bag - the bag of values
name - the name assigned to the bag
help - help message for bag
