MatPartitioningType

 String with the name of a PETSc matrix partitioning or the creation function with an optional dynamic library name, for example http://www.mcs.anl.gov/petsc/lib.a:partitioningcreate()

Synopsis

#define MAT_PARTITIONING_CURRENT  "current"
