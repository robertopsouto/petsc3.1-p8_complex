MatInitializePackage

 This function initializes everything in the Mat package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to MatCreate() when using static libraries.

Synopsis

PetscErrorCode PETSCMAT_DLLEXPORT MatInitializePackage(const char path[])

Input Parameter

path - The dynamic library path, or PETSC_NULL

Keywords

Mat, initialize, package

See Also
