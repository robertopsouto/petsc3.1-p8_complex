MATMPISBAIJ

 MATMPISBAIJ = "mpisbaij" - A matrix type to be used for distributed symmetric sparse block matrices, based on block compressed sparse row format. Only the upper triangular portion of the matrix is stored.

Options Database Keys

-mat_type mpisbaij -sets the matrix type to "mpisbaij" during a call to MatSetFromOptions()
