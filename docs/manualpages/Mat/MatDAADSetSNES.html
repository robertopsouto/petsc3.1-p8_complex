MatDAADSetSNES

 Tells the matrix what SNES it is using for the base U.

Synopsis

#include "petscda.h"    
#include "petscsnes.h"  
PetscErrorCode PETSCMAT_DLLEXPORT MatDAADSetSNES(Mat A,SNES snes)
Collective on Mat and SNES

Input Parameters

mat - the matrix
snes - the SNES

Notes: this is currently turned off for Fortran usage
