MatZeroEntries

 Zeros all entries of a matrix. For sparse matrices this routine retains the old nonzero structure.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatZeroEntries(Mat mat)
Collective on Mat

Input Parameters

mat -the matrix
