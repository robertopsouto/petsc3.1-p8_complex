MatScale

 Scales all elements of a matrix by a given number.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatScale(Mat mat,PetscScalar a)
Collective on Mat

Input Parameters

mat - the matrix to be scaled
a - the scaling value

Output Parameter

mat -the scaled matrix
