MatIsHermitian

 Test whether a matrix is Hermitian

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatIsHermitian(Mat A,PetscReal tol,PetscTruth *flg)
Collective on Mat

Input Parameter

A - the matrix to test
tol - difference between value and its transpose less than this amount counts as equal (use 0.0 for exact Hermitian)

Output Parameters

flg -the result

C++ variants

PetscTruth MatIsHermitian(Mat A)->MatIsHermitian(A,0,&t); return t;
