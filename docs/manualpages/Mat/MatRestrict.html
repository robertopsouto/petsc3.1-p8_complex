MatRestrict

 y = A*x or A'*x

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatRestrict(Mat A,Vec x,Vec y)
Collective on Mat

Input Parameters

mat - the matrix
x,y - the vectors

Notes

This allows one to use either the restriction or interpolation (its transpose) matrix to do the restriction
