MatTranspose

 Computes an in-place or out-of-place transpose of a matrix.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatTranspose(Mat mat,MatReuse reuse,Mat *B)
Collective on Mat

Input Parameter

mat - the matrix to transpose
reuse - store the transpose matrix in the provided B

Output Parameters

B -the transpose

C++ variants

Mat MatTranspose(Mat A)->MatTranspose(A,MAT_INITIAL_MATRIX,&t); return t;

Notes

If you pass in &mat for B the transpose will be done in place

See Also

MatMultTranspose(), MatMultTransposeAdd(), MatIsTranspose(), MatReuse
