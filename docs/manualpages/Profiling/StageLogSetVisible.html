StageLogSetVisible

 This function determines whether a stage is printed during PetscLogPrintSummary()

Synopsis

#include "petscsys.h"   
PetscErrorCode PETSC_DLLEXPORT StageLogSetVisible(StageLog stageLog, int stage, PetscTruth isVisible)
Not Collective

Input Parameters

stageLog - The StageLog
stage - The stage to log
isVisible - The visibility flag, PETSC_TRUE for printing, otherwise PETSC_FALSE (default is PETSC_TRUE)

Database Options

-log_summary -Activates log summary

Keywords

log, visible, stage

See Also

StageLogGetVisible(), StageLogGetCurrent(), StageLogRegister(), PetscLogGetStageLog()
