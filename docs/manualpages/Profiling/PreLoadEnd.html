PreLoadEnd

 End a segment of code that may be preloaded (run twice) to get accurate timings

Synopsis

void PreLoadEnd(void);
Not Collective

Usage

     PreLoadBegin(PETSC_TRUE,"first stage);
       lines of code
       PreLoadStage("second stage");
       lines of code
     PreLoadEnd();

Notes: only works in C/C++ not fortran
