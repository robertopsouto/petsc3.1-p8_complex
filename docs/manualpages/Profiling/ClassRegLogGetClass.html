ClassRegLogGetClass

 This function returns the class corresponding to a given cookie.

Synopsis

#include "petscsys.h"   
PetscErrorCode ClassRegLogGetClass(ClassRegLog classLog, PetscCookie cookie, int *oclass)
Not Collective

Input Parameters

classLog - The ClassRegLog
cookie - The cookie

Output Parameter

oclass -The class id

Keywords

log, class, register

See Also

PetscCookieRegister(), PetscLogObjCreateDefault(), PetscLogObjDestroyDefault()
