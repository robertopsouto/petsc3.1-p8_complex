EventPerfLogSetVisible

 This function determines whether an event is printed during PetscLogPrintSummary()

Synopsis

#include "petscsys.h"   
PetscErrorCode EventPerfLogSetVisible(EventPerfLog eventLog, PetscLogEvent event, PetscTruth isVisible)
Not Collective

Input Parameters

eventLog - The EventPerfLog
event - The event to log
isVisible - The visibility flag, PETSC_TRUE for printing, otherwise PETSC_FALSE (default is PETSC_TRUE)

Database Options

-log_summary -Activates log summary

Keywords

log, visible, event

See Also

EventPerfLogGetVisible(), EventRegLogRegister(), StageLogGetEventLog()
