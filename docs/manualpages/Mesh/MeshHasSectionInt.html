MeshHasSectionInt

 Determines whether this mesh has a SectionInt with the given name.

Synopsis

#include "petscmesh.h"   
PetscErrorCode MeshHasSectionInt(Mesh mesh, const char name[], PetscTruth *flag)
Not Collective

Input Parameters

mesh - The Mesh object
name - The section name

Output Parameter

flag -True if the SectionInt is present in the Mesh

Keywords

mesh, elements
