MeshCreateVector

 Creates a global vector matching the input section

Synopsis

#include "petscmesh.h"   
PetscErrorCode PETSCDM_DLLEXPORT MeshCreateVector(Mesh mesh, SectionReal section, Vec *vec)
Collective on Mesh

Input Parameters

mesh - the Mesh
section - the Section

Output Parameter

vec -the global vector

Notes: The vector can safely be destroyed using VecDestroy().

See Also

MeshDestroy(), MeshCreate(), MeshGetGlobalIndices()
