MeshHasSectionReal

 Determines whether this mesh has a SectionReal with the given name.

Synopsis

#include "petscmesh.h"   
PetscErrorCode MeshHasSectionReal(Mesh mesh, const char name[], PetscTruth *flag)
Not Collective

Input Parameters

mesh - The Mesh object
name - The section name

Output Parameter

flag -True if the SectionReal is present in the Mesh

Keywords

mesh, elements
