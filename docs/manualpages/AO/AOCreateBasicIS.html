AOCreateBasicIS

 Creates a basic application ordering using two index sets.

Synopsis

#include "petscao.h"   
PetscErrorCode PETSCDM_DLLEXPORT AOCreateBasicIS(IS isapp,IS ispetsc,AO *aoout)
Collective on IS

Input Parameters

isapp - index set that defines an ordering
ispetsc - index set that defines another ordering (may be PETSC_NULL to use the natural ordering)

Output Parameter

aoout -the new application ordering

Options Database Key

-ao_view -call AOView() at the conclusion of AOCreateBasicIS()

Notes: the index sets isapp and ispetsc must contain the all the integers 0 to napp-1 (where napp is the length of the index sets) with no duplicates; that is there cannot be any "holes"

Keywords

AO, create

See Also

AOCreateBasic(), AODestroy()
